# Icons

Display semantic vector graphics in SVG format.

## Installation

Install the package in your project directory.

```bash
bun add @geist-ui/icons
```

## Import

```jsx
import { Github } from '@geist-ui/icons'

const MyComponent = () => <Github />
```
