# Typography

Geist uses a clean, sans-serif stack that prioritizes legibility and technical precision.

**Font Family:** `Inter`, `-apple-system`, `BlinkMacSystemFont`, `"Segoe UI"`, `Roboto`, `Oxygen`, `Ubuntu`, `Cantarell`, `"Fira Sans"`, `"Droid Sans"`, `"Helvetica Neue"`, `sans-serif`

## Weights

*   **Regular (400)**: Used for body text and general labels.
*   **Medium (500)**: Used for sub-headings and emphasized text.
*   **Semi-Bold (600)**: Used for main headings and interactive elements.
*   **Bold (700)**: Used for high-level page titles.
