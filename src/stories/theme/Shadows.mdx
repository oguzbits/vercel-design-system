## Usage Guidelines

- **Small**: Used for static components that need a subtle lift from the background, like small cards or tooltips.
- **Medium**: Standard elevation for primary interactive elements like Buttons and Dropdowns.
- **Large**: High elevation for critical UI overlays like Modals and Popovers.
- **Brand**: Special elevation for primary call-to-action focuses.
