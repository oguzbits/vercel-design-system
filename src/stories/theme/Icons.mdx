# Icons

Our iconography system is designed to be simple, consistent, and functional. We use `@geist-ui/icons` for a cohesive look across all applications.

## Usage

Icons should be used to provide visual context and improve scannability. They should NOT be used as the sole means of communication for critical actions.

```jsx
import { Github } from "@geist-ui/icons";

const MyComponent = () => <Github color="black" size={24} />;
```

## Sizes

Icons are available in three standard sizes:

- `16px`: For dense UI and labels.
- `24px`: Default size for actions and nav.
- `32px`: For empty states and hero sections.
