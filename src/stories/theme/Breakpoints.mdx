# Breakpoints

Geist follows a mobile-first responsive strategy. Our breakpoints are designed to handle everything from narrow mobile devices to ultra-wide monitors.

## Standard Breakpoints

| Name | Size              | Device Category                 |
| :--- | :---------------- | :------------------------------ |
| `xs` | `< 650px`         | Mobile phones.                  |
| `sm` | `650px - 900px`   | Large phones and small tablets. |
| `md` | `900px - 1280px`  | Tablets and small laptops.      |
| `lg` | `1280px - 1920px` | Standard desktops.              |
| `xl` | `> 1920px`        | Ultra-wide monitors.            |

## Media Queries

You can use the built-in Geist breakpoints in your CSS:

```css
@media only screen and (max-width: 650px) {
  /* Mobile styles */
}
```

In React, use the `useMediaQuery` hook for conditional rendering:

```jsx
const isMobile = useMediaQuery("xs");
```
