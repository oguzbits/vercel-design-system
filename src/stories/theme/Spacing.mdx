# Spacing

The Geist spacing system is based on an 8px / 4px modular grid. Consistency in spacing is vital for creating a balanced and rhythmically pleasing interface.

## Spacing Scale

Our scale ranges from `4px` (half-unit) to `80px` (massive).

<div style={{ marginTop: '2rem' }}>

      <div style={{ width: '80px', fontSize: '0.8rem', color: 'var(--accents-5)' }}>{size}px</div>

## Layout Grid

For large-scale layouts, we recommend using a standard 12-column grid or CSS Flexbox/Grid with the established spacing units.

**Recommended Margins:**
*   **Mobile**: 16px
*   **Tablet**: 24px
*   **Desktop**: 48px
*   **Wide**: 64px
